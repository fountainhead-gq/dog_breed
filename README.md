# Dog Breed Identification

Kaggle的狗狗种类识别，[Dog Breed Identification](https://www.kaggle.com/c/dog-breed-identification/data)



### 模型融合

- Xception 

- InceptionV3 

- InceptionResNetV2


### 结果



| model                                    | epochs | result |
| ---------------------------------------- | ------ | ------ |
| InceptionV3 + Xception + inceptionResNetV2 | 15     | 0.46   |
| inceptionResNetV2                        | 20     | 0.7    |
